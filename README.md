# LPem
Repository ini berisi kumpulan latihan, panduan lenkap dan contoh kode untuk mempelajari dasar-dasar pemrograman menggunakan Python. Proyek ini dibuat sebagai dokumentasi Juorney pemrograman , serta sebagai referensi terbuka bagi siapa pun yang ingin memahami konsep dasar pemrograman secara praktis dan mudah dipahami.
